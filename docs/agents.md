# AI Development Agents

This project uses AI-assisted development.

## Agent Responsibilities

### Architecture Agent

Responsible for:

* system design
* API architecture
* database schema

### Backend Agent

Responsible for:

* API implementation
* authentication
* database queries

### Testing Agent

Responsible for:

* writing tests
* verifying API responses
* debugging issues

---

## Agent Rules

1. Follow modular architecture.
2. Do not modify frontend without approval.
3. Maintain API documentation.
4. Always test endpoints before committing.
